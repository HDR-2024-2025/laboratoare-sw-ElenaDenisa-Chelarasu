Log file analizer usage:

```
# Show 4xx errors
python log_analyzer.py 1

# Show URL statistics
python log_analyzer.py 2

# Watch logs in real-time
python log_analyzer.py 3

# Specify a different log directory
python log_analyzer.py 2 --log-dir /path/to/logs
```
